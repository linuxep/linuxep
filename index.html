Get Started

LEPD | Linux Easy Profiling Daemon

LEPD is a daemon running on the server to be monitored

Here's is how to install and run LEPD


# Clone the latest LEPD code from Github
    git clone https://github.com/linuxep/lepd

# Build the binary
    - For X86（libev-dev is required）:
      ./build_x86.sh
    - For ARM（arm-linux-gnueabi-gcc is required）:
      ./build_arm.sh

# Run LEPD with root permission

LEPV | Linux Easy Profiling Viewer

LEPV is a web-based viewer to present profiling data visually in real time


# Clone the latest LEPV code from Github
    git clone https://github.com/linuxep/lepv.git
    
# Run a script from root directory to build the Docker image
    ./buildDockerImage.sh
    
# Start a Docker container
    ./runDockerContainer.sh
    
# Now Open up your favorite browser and navigate to:
    http://localhost:8889
